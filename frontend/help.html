QuizBuilder AI – User Guide

Create, compare, and approve AI‑generated practice questions – fast.

Tip: Press Enter to generate. Inside the notes box use Cmd/⌘ + Enter (Mac) or Ctrl + Enter (Windows/Linux).

1) Generate

  • Select Exam, Language, Type, and Difficulty.
  • (Optional) Add Notes to guide the models.
  • Choose the models (GPT / Gemini / Anthropic / xAI).
  • Click Generate (or press Enter).

2) Review the Selection Round

After generation, a Selection Round Results panel appears above the list:

  • Tabs show each model’s candidate – ordered by final rank (winner first).
  • The STATS tab shows points, mean score, and per‑evaluator ranks.
  • Each candidate card includes an Evaluations table with per‑engine feedback.

3) Improve (optional)

  • Enter tutor feedback in the card’s comment box.
  • Click Send Comment & Improve to revise the item.

4) Approve / Delete

  • Click Approve & Add to CSV to mark the question approved and append it to today’s CSV.
  • Use Delete to move a question to the Deleted tab. You can restore it later.

5) Export

  • Click Download CSV to open a file picker of all CSVs in the data directory.
  • The app writes one CSV per day (UTC). Pick any file to download.

Troubleshooting

  • Don’t see new UI changes? Hard refresh (Cmd/Shift/R on macOS).
  • Missing files in the picker? Ensure the server can write to the data directory.
  • API errors? Check the server logs and confirm your API keys/credits.
